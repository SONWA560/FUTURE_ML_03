# Frontend

Next.js (App Router) + shadcn/ui, themed with the tweakcn "darkmatter" theme.

See the [project root README](../README.md) for setup instructions and how the whole system fits together.

```bash
pnpm install
pnpm dev
```

Requires the backend API running (see `../backend/README.md` or the root README) and a `.env.local` with `NEXT_PUBLIC_API_URL` (see `.env.example`).
